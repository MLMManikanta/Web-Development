CSS Transform Property

The transform property visually manipulates an element without affecting the page layout. It supports 2D and 3D movement, rotation, scaling, and skewing.

1. Transform Syntax

transform: function(value);

2. 2D Transform Functions

2.1 translate(x, y) — Moves an element in 2D space

transform: translate(40px, 15px);
translate()

2.2 translateX(x) — Moves horizontally

transform: translateX(60px);
translateX()

2.3 translateY(y) — Moves vertically

transform: translateY(40px);
translateY()

2.4 scale(x, y) — Resizes element

transform: scale(1.4);
scale()

2.5 scaleX(x) — Stretches horizontally

transform: scaleX(1.6);
scaleX()

2.6 scaleY(y) — Stretches vertically

transform: scaleY(1.6);
scaleY()

2.7 rotate(angle) — Rotates element

transform: rotate(30deg);
rotate()

2.8 skew(x-angle, y-angle) — Slants element

transform: skew(15deg, 8deg);
skew()

2.9 skewX(angle) — Slants horizontally

transform: skewX(25deg);
skewX()

2.10 skewY(angle) — Slants vertically

transform: skewY(25deg);
skewY()

2.11 matrix() — Combines scale/translate/skew

transform: matrix(1,0.2,0.3,1,25,15);
matrix()

3. 3D Transform Functions

3.1 translateZ(z) — Moves forward/backward

transform: translateZ(80px);
translateZ()

3.2 rotateX(angle) — Rotates around X-axis

transform: rotateX(45deg);
rotateX()

3.3 rotateY(angle) — Rotates around Y-axis

transform: rotateY(45deg);
rotateY()

3.4 rotateZ(angle) — Rotates around Z-axis

transform: rotateZ(45deg);
rotateZ()

3.5 scaleZ — Scales along Z-axis

transform: scaleZ(2);
scaleZ()

3.6 matrix3d() — Full 3D transformation

transform: matrix3d(...);
matrix3d()

4. transform-origin — Sets pivot point

transform-origin: left top;
transform-origin

5. transform-style — Enables nested 3D

transform-style: preserve-3d;
preserve-3d

6. perspective — Adds camera depth

perspective: 300px;
perspective

7. perspective-origin — Controls vanishing point

perspective-origin: left top;
perspective-origin
