1. Transform Syntax

2. 2D Transform Functions

transform: translateY(40px);
translateY()

2.4 scale(x, y) — Resizes element

transform: scale(1.4);
scale()

transform: scaleX(1.6);
scaleX()

2.6 scaleY(y) — Stretches vertically

transform: scaleY(1.6);
scaleY()

rotate()

2.10 skewY(angle) — Slants vertically

CSS Box-Shadow Property

The box-shadow property adds one or more drop shadows around an element. It supports horizontal/vertical offsets, blur, spread, color, inset, and multiple shadows.

1. box-shadow Syntax

box-shadow: offset-x offset-y blur-radius spread-radius color;

Understanding box-shadow Parameters

The box-shadow property uses 5 main values. Each value changes the placement, size, and appearance of the shadow.

1. offset-x — Horizontal Shadow Position

Moves the shadow left or right. Positive → Moves right | Negative → Moves left 

box-shadow: 20px 0 10px rgba(0,0,0,0.3);
/* 20px = offset-x */
offset-x

2. offset-y — Vertical Shadow Position

Moves the shadow up or down. Positive → Down | Negative → Up 

box-shadow: 0 20px 10px rgba(0,0,0,0.3);
/* 20px = offset-y */
offset-y

3. blur-radius — Softness of Shadow

Controls how blurry the shadow looks. Higher value = softer shadow. Cannot be negative. 

box-shadow: 0 0 20px rgba(0,0,0,0.4);
/* 20px = blur-radius */
blur-radius

4. spread-radius — Expands or Shrinks Shadow Size

Controls how large the shadow grows. Positive → Shadow expands | Negative → Shadow shrinks 

box-shadow: 0 0 10px 15px rgba(0,0,0,0.3);
/* 15px = spread-radius */
spread-radius

5. color — Shadow Color

Determines the shadow’s color and transparency. Supports: color name, HEX, RGB, RGBA, HSL. 

box-shadow: 0 0 15px rgba(59,130,246,0.7);
/* rgba(59,130,246,0.7) = color */
color

2. Basic Shadows

2.1 Normal Shadow — Basic drop shadow

box-shadow: 4px 4px 10px rgba(0,0,0,0.3);
Basic Shadow

2.2 Soft Shadow — Bigger blur for a softer look

box-shadow: 0 6px 18px rgba(0,0,0,0.25);
Soft Shadow

2.3 Sharp Shadow — Hard edges with low blur

box-shadow: 5px 5px 0 rgba(0,0,0,0.4);
Sharp Shadow

3. Spread Shadows

3.1 Spread Shadow — Expands shadow size

box-shadow: 0 6px 10px 5px rgba(0,0,0,0.3);
Spread Shadow

4. Inset Shadows

4.1 Inset Shadow — Shadow inside the element

box-shadow: inset 4px 4px 10px rgba(0,0,0,0.4);
Inset Shadow

5. Multiple Shadows

5.1 Stacked Shadows — Apply layered shadow effects

box-shadow:
    0 0 5px rgba(0,0,0,0.3),
    0 8px 20px rgba(0,0,0,0.15),
    0 12px 35px rgba(0,0,0,0.1);
Multiple Shadows

6. Colored Shadows

6.1 Light Blue Shadow — Colorful UI cards

box-shadow: 0 8px 18px rgba(59,130,246,0.4);
Blue Shadow

6.2 Red Glow Shadow — Glow effect

box-shadow: 0 0 15px rgba(239,68,68,0.7);
Glow Shadow

7. Neumorphism Effect

7.1 Soft UI — Dual shadows

box-shadow:
    8px 8px 16px rgba(0,0,0,0.2),
    -8px -8px 16px rgba(255,255,255,0.8);
Neumorphism

8. Combined Inner & Outer Shadow

box-shadow:
    inset 4px 4px 8px rgba(0,0,0,0.25),
    0 8px 15px rgba(0,0,0,0.2);
Inner + Outer

CSS Animations

The animation property allows elements to move, rotate, fade, grow, or transform over time using @keyframes. Animations can loop, reverse, pause, and control how styles apply before and after animation.

1. @keyframes — Defining animation frames

@keyframes example {
    from { opacity: 0; }
    to   { opacity: 1; }
}

2. animation-name — Which animation to run

2.1 Fade in

animation-name: fadeIn;
Fade In

3. animation-duration — How long animation runs

3.1 2 second animation

animation-duration: 2s;
2 Seconds

4. animation-timing-function — Speed curve

4.1 ease

animation-timing-function: ease;
ease

4.2 linear

animation-timing-function: linear;
linear

4.3 ease-in

animation-timing-function: ease-in;
ease-in

4.4 ease-out

animation-timing-function: ease-out;
ease-out

4.5 cubic-bezier()

animation-timing-function:
    cubic-bezier(0.68,-0.55,0.27,1.55);
cubic-bezier

5. animation-delay — Wait before animation starts

5.1 Delay of 1 second

animation-delay: 1s;
Delay

6. animation-iteration-count — Number of times animation repeats

6.1 Loop 3 times

animation-iteration-count: 3;
3 Loops

6.2 Infinite Loop

animation-iteration-count: infinite;
Infinity Loop

7. animation-direction — Direction animation runs

  • normal — forward
  • reverse — backward
  • alternate — forward then backward
  • alternate-reverse — backward then forward

7.1 alternate

animation-direction: alternate;
Alternate

8. animation-fill-mode — Apply styles before/after animation

  • none — no style retained
  • forwards — keep final style
  • backwards — apply starting style during delay
  • both — apply both

8.1 forwards

animation-fill-mode: forwards;
forwards

9. animation-play-state — Play/Pause animation

9.1 Paused on Hover

animation-play-state: paused;
Hover to Pause

10. Multiple Animations

10.1 Move + Fade

animation:
    slide 2s ease-in-out infinite,
    fade 3s ease infinite;
Move + Fade

CSS Clip-Path

The clip-path property clips an element into custom shapes such as circles, polygons, ellipses, triangles, diamonds, and more. It is commonly used for creative UI effects, image shapes, and masking.

1. clip-path Syntax

clip-path: shape(values);

2. Basic Shapes

2.1 circle()

clip-path: circle(50%);

2.2 ellipse()

clip-path: ellipse(60% 40%);

2.3 inset()

clip-path: inset(20px 30px);

3. Polygon Shapes

3.1 Triangle

3.2 Diamond

clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%);

3.3 Hexagon

    25% 0%, 75% 0%,
