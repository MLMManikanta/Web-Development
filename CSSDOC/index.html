CSS (Cascading Style Sheets)

CSS is a styling language used to design and control the layout of HTML webpages. It allows you to change colors, fonts, spacing, backgrounds, borders, and the overall appearance of web pages.

1. Inline CSS

Inline CSS is written directly within an HTML tag using the style attribute. It is useful for applying unique styles to a single element but not recommended for large projects.

This paragraph uses Inline CSS. 

<p style="color: orangered; font-size: 18px;">This is Inline CSS</p>

2. Internal CSS

Internal CSS is written inside a <style> tag within the <head> section of the HTML document. It allows you to apply styles to the entire page in one place.

This paragraph is styled with Internal CSS.


<style>
  .internal-css-example {
    color: darkblue;
    background-color: #eef;
    padding: 10px;
    border-radius: 8px;
  }
</style>

3. External CSS

External CSS is written in a separate file (for example, css.css) and linked to the HTML document using the <link> tag. It’s the best practice for maintaining clean and reusable code.

This paragraph is styled using External CSS.


<link rel="stylesheet" href="css.css">

h1 {
  color: green;
}
p {
  font-size: 18px;
}

4. What is a CSS Selector?

A CSS Selector is a pattern used to target specific HTML elements so you can apply styles to them.

Example: p { color: purple; font-weight: bold; }

In this example, the p selector targets all <p> tags in the document.

Types of CSS Selectors with Examples

1. Element Selector

This styles all <p> elements.

p { color: darkblue; }

This paragraph is styled using an element selector.

2. Class Selector

Targets elements with a specific class name.

.highlight { background-color: yellow; }

This paragraph uses a class selector (.highlight).

3. ID Selector

Targets a unique element with an ID.

#unique { color: crimson; }

This text is styled using an ID selector (#unique).

4. Group Selector

Styles multiple elements at once, separated by commas.

h1, h2, p { font-family: Arial; }

This paragraph shares styles with h1 and h2 via grouping.

5. Descendant Selector

Targets elements nested inside others.

div p { color: green; }

This paragraph is inside a <div> and styled with a descendant selector.

Targets direct child elements only.

div > span { color: darkorange; }

Targets the element immediately following another.

CSS Reset and Box Model

CSS Reset:

* { margin: 0; padding: 0; box-sizing: border-box; }

This removes default spacing and ensures consistent box sizing across all browsers. It makes layout calculations easier and more predictable.

Margin and Padding Shorthand Formats

Both margin and padding can accept one, two, three, or four values:

  • 1 value: applies to all four sides
    margin: 20px; → top = right = bottom = left = 20px
  • 2 values: first = top & bottom, second = left & right
    padding: 10px 30px; → top/bottom = 10px, left/right = 30px
  • 3 values: first = top, second = left & right, third = bottom
    margin: 5px 15px 10px; → top = 5px, left/right = 15px, bottom = 10px
  • 4 values: top → right → bottom → left (clockwise order)
    padding: 5px 10px 15px 20px;
    → top = 5px, right = 10px, bottom = 15px, left = 20px

5. Common CSS Properties Explained

Below are the most commonly used CSS properties with examples and their corresponding code:

  1. font-size – Controls how big the text appears
    Small Font | Medium Font | Large Font

    p { font-size: 18px; }
  2. color – Changes the text color
    Red Color | Blue Color | Green Color

    p { color: blue; }
  3. background-color – Sets the background behind an element
    Yellow Background | Light Blue Background | Light Gray Background

    div { background-color: lightgray; }
  4. width & height – Controls the size of the box
    Small Box
    Medium Box


    div { width: 150px; height: 100px; }
  5. padding – Adds inner spacing inside a box
    Box with Padding


    div { padding: 20px; }
  6. margin – Adds outer spacing outside an element
    Box with Margin


    div { margin: 20px; }
  7. border – Adds a border around an element
    Box with Border


    div { border: 2px solid blue; }
  8. border-radius – Rounds the corners of an element
    Rounded Corners


    div { border-radius: 10px; }
  9. font-weight – Makes text bold or normal
    Normal Text | Bold Text

    p { font-weight: bold; }
  10. text-align – Aligns text inside an element
    Left Aligned
    Center Aligned
    Right Aligned


    p { text-align: center; }
