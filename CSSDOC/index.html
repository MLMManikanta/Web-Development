CSS (Cascading Style Sheets)

CSS is a styling language used to design and control the layout of HTML webpages. It allows you to change colors, fonts, spacing, backgrounds, borders, and the overall appearance of web pages.

1. Inline CSS

Inline CSS is written directly within an HTML tag using the style attribute. It is useful for applying unique styles to a single element but not recommended for large projects.

This paragraph uses Inline CSS. 

<p style="color: orangered; font-size: 18px;">This is Inline CSS</p>

2. Internal CSS

Internal CSS is written inside a <style> tag within the <head> section of the HTML document. It allows you to apply styles to the entire page in one place.

This paragraph is styled with Internal CSS.


<style>
  .internal-css-example {
    color: darkblue;
    background-color: #eef;
    padding: 10px;
    border-radius: 8px;
  }
</style>

3. External CSS

External CSS is written in a separate file (for example, css.css) and linked to the HTML document using the <link> tag. It‚Äôs the best practice for maintaining clean and reusable code.

This paragraph is styled using External CSS.


<link rel="stylesheet" href="css.css">

h1 {
  color: green;
}
p {
  font-size: 18px;
}

A CSS Selector is a pattern used to target specific HTML elements so you can apply styles to them.

Example: p { color: purple; font-weight: bold; }

5. CSS Margin, Padding, and Outline Properties

The margin, padding, and outline properties define the space around elements and their outer visual styling. While margin and padding control spacing, outlines are decorative lines drawn outside the border.

1. Margin and Padding Shorthand Formats

Both margin and padding can accept one, two, three, or four values:

  • 1 value: applies to all four sides
    margin: 20px; ‚Üí top = right = bottom = left = 20px
  • 2 values: first = top & bottom, second = left & right
    padding: 10px 30px; ‚Üí top/bottom = 10px, left/right = 30px
  • 3 values: first = top, second = left & right, third = bottom
    margin: 5px 15px 10px; ‚Üí top = 5px, left/right = 15px, bottom = 10px
  • 4 values: top ‚Üí right ‚Üí bottom ‚Üí left (clockwise order)
    padding: 5px 10px 15px 20px;
    ‚Üí top = 5px, right = 10px, bottom = 15px, left = 20px
Box with margin and padding

2. Outline Property

The outline property draws a line outside an element‚Äôs border ‚Äî often used for highlighting or accessibility focus styles. Unlike borders, outlines do not affect the element‚Äôs size or box layout.

Key Differences Between Border and Outline

  • Outline lies outside the border, not inside the box.
  • Outline does not take space (it‚Äôs drawn over the box).
  • Outline can‚Äôt be applied to specific sides (top, right, bottom, left).

2.1 outline-style

Defines the line pattern of the outline. Common values include:

  • solid ‚Äì a single solid line
  • dashed ‚Äì short dashes
  • dotted ‚Äì small round dots
  • double ‚Äì two parallel lines
  • groove, ridge, inset, outset ‚Äì 3D effects based on light/dark shading

Specifies the color of the outline. You can use color names, HEX, RGB, RGBA, or HSL values.

2.3 outline-width

Sets the thickness of the outline. You can use keywords or pixel values:

  • thin
  • medium
  • thick
  • px or em for custom sizes

2.4 outline-offset

The outline-offset property adds space between an element‚Äôs border and its outline, creating a visible gap for better focus or hover effects.

You can combine all outline properties into one shorthand declaration:

8. Universal Selector and Box-Sizing Property

The * symbol is called the Universal Selector. It selects every element on the webpage and applies the defined CSS properties to all of them.

This removes default spacing and ensures consistent box sizing across all browsers. It makes layout calculations easier and more predictable. 


* {
    margin: 0;
    padding: 0;
    box-sizing: border-box;
}

Explanation:

  • margin: 0; ‚Üí Removes the outer spacing around all elements.
  • padding: 0; ‚Üí Removes the inner spacing inside all elements.
  • box-sizing: border-box; ‚Üí Includes padding and border within the total width and height of an element.

This is used as a CSS Reset to make layouts consistent across browsers.

10. Colors in CSS

Colors play a vital role in the design and appearance of a webpage. CSS allows you to set colors for text, backgrounds, borders, and other visual elements using multiple formats.

Ways to Define Colors in CSS

  1. 1. Named Colors
    CSS supports 140 standard color names such as red, blue, green, tomato, gold, etc.

    Tomato
    Sky Blue
    Lime Green

  2. 2. HEX Values
    A hexadecimal color uses a combination of six characters representing red, green, and blue components.

  3. 3. RGB Colors
    Uses the rgb() function to define colors by specifying red, green, and blue values (0‚Äì255).

  4. 4. RGBA Colors
    Similar to RGB but includes an alpha channel for transparency. Alpha values range from 0 (fully transparent) to 1 (fully opaque).

    RGBA(255, 99, 71, 0.5)
    RGBA(255, 99, 71, 0.2)

  5. 5. HSL Colors
    The hsl() function defines color using hue, saturation, and lightness.

Useful Tips:

  • Use online color pickers (like Adobe Color) to choose palettes.
  • Use consistent color schemes for accessibility and readability.
