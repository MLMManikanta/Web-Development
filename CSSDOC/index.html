CSS (Cascading Style Sheets)

CSS is a styling language used to design and control the layout of HTML webpages. It allows you to change colors, fonts, spacing, backgrounds, borders, and overall page appearance.

1. Inline CSS

Inline CSS is written directly inside an HTML tag using the style attribute.

Example (this paragraph uses inline CSS):
<p style="color: red; font-size: 18px;">This is Inline CSS</p>

2. Internal CSS

Internal CSS is written inside the <style> tag in the <head> section.

Example (this paragraph is styled with internal CSS):
<style>
  .internal-css-example { color: darkblue; background-color: #eef; }
</style>

3. External CSS

External CSS is written in a separate file (like css.css) and linked using
the <link rel="stylesheet" href="css.css"> tag.

Example CSS file content:
h1 { color: green; }
p { font-size: 18px; }

4. What is a CSS Selector?

A CSS Selector is a keyword or pattern used to select the HTML elements you want to style.

Example:
p { color: purple; font-weight: bold; }

In this example, p is the selector that targets all <p> tags.

margin: 0px; → Removes all outer spacing (margin) around elements.
padding: 0px; → Removes all inner spacing (padding) inside elements.
box-sizing: border-box; → Ensures that an element’s total width and height include its padding and border, making layout calculations easier and more predictable.

This combination is often used at the top of every stylesheet as a CSS Reset to ensure consistent rendering across all browsers.

5. Common CSS Properties Explained

Below are the most commonly used CSS properties with examples and their corresponding code:

  1. font-size – Controls how big the text appears
    Small Font | Medium Font | Large Font

    p { font-size: 18px; }
  2. color – Changes the text color
    Red Color | Blue Color | Green Color

    p { color: blue; }
  3. background-color – Sets the background behind an element
    Yellow Background | Light Blue Background | Light Gray Background

    div { background-color: lightgray; }
  4. width & height – Controls the size of the box
    Small Box
    Medium Box


    div { width: 150px; height: 100px; }
  5. padding – Adds inner spacing inside a box
    Box with Padding


    div { padding: 20px; }
  6. margin – Adds outer spacing outside an element
    Box with Margin


    div { margin: 20px; }
  7. border – Adds a border around an element
    Box with Border


    div { border: 2px solid blue; }
  8. border-radius – Rounds the corners of an element
    Rounded Corners


    div { border-radius: 10px; }
  9. font-weight – Makes text bold or normal
    Normal Text | Bold Text

    p { font-weight: bold; }
  10. text-align – Aligns text inside an element
    Left Aligned
    Center Aligned
    Right Aligned


    p { text-align: center; }

6. CSS Text Properties

Text properties help you control how text looks on a webpage — including size, case, weight, spacing, decoration, and alignment.

  1. font-size – Controls how big or small the text appears
    Small Font Size
    Medium Font Size
    Large Font Size

    p { font-size: 20px; }
  2. text-transform – Changes the capitalization of text
    uppercase text
    LOWERCASE TEXT
    capitalize this sentence

    p { text-transform: uppercase; }
  3. font-weight – Controls how thick or bold the text looks
    Light Weight
    Normal Weight
    Bold Weight

    p { font-weight: bold; }
  4. font-style – Applies italic or oblique slant to text
    Italic Style
    Oblique Style

    p { font-style: italic; }
  5. spacing – Adds space between letters or words
    Letter Spacing Example
    Word Spacing Example

    p { letter-spacing: 3px; }
  6. text-decoration – Adds underline, overline, or strikethrough
    Underline Example
    Line Through Example
    Overline Example

    p { text-decoration: underline; }
  7. text-align – Controls horizontal text alignment
    This text is left aligned
    This text is center aligned
    This text is right aligned

    p { text-align: center; }
  8. line-height – Controls vertical spacing between lines of text
    This is an example of text
    with small line height.

    This is an example of text
    with large line height.

    p { line-height: 32px; }
