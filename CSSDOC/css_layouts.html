CSS Layout

CSS Layout controls how elements are arranged on a webpage — including their display behavior, alignment, and positioning. The foundation of any web design starts with understanding the display property.

1. The display Property

The display property defines how an element behaves in the document flow. It determines whether an element acts as a block, inline, or inline-block element — or if it’s hidden completely.

1 Block Elements

Block-level elements always start on a new line and stretch to fill the entire width. Common examples include <div>, <p>, and <section>.

Block 1
Block 2

                div {
                    display: block;
                }

2 Inline Elements

Inline elements only take up as much width as their content needs. They sit next to each other on the same line — like <span>, <a>, and <strong>.

Inline 1 Inline 2 

            span {
                display: inline;
            }

3 Inline-Block Elements

The inline-block display allows elements to behave like inline elements, but still respect width and height like block elements.

Box 1
Box 2

            div {
                display: inline-block;
                width: 120px;
                height: 80px;
            }

4 None

When display: none; is applied, the element is completely removed from the page layout.

You can hide or show me!

            div {
                display: none; /* completely hides element */
            }

5 display: flow-root

The flow-root value creates a new Block Formatting Context (BFC). It helps contain floated elements inside their parent automatically, preventing layout collapse without using the old clearfix method.

Floating Box

This container uses display: flow-root to wrap floated children correctly.


.container {
    display: flow-root;
}
.float-box {
    float: left;
    width: 120px;
    height: 80px;
    background: #60a5fa;
    color: white;
    text-align: center;
    line-height: 80px;
    border-radius: 6px;
    margin-right: 10px;
}

6 display: table

Using display: table makes elements behave like an HTML table, with children acting as table-row and table-cell. This allows table-like layouts without using actual table tags.

7 display: list-item

The list-item value gives any element a list marker (like a bullet or number). It’s usually used with <li> tags but works on other tags too.

Custom Item 1
Custom Item 2

    display: list-item;
    list-style: square inside;
}

8 display: contents

The contents value makes the element itself disappear, but keeps its child elements visible as if they were direct children of the parent.

Child 1 Child 2 

.wrapper {
    display: contents;
}

9 display: run-in

The run-in display makes an element behave as inline or block depending on the context. It’s a rare and poorly supported layout mode.

Run-in Heading

This paragraph follows a run-in heading.


.runin-heading {
    display: run-in;
}

The ruby display type is used in East Asian typography for adding pronunciation or annotation above/below characters.

    display: ruby;
}

    display: ruby-text;

2. CSS Flexbox Layout

The Flexible Box Layout (Flexbox) is a modern CSS layout model for arranging items in one direction — horizontally or vertically. It allows flexible alignment, spacing, and ordering of elements, making it easier to create responsive designs.

2.1 display: flex

The display: flex; property turns a container into a Flexbox layout. All direct children become flex items that can be positioned and aligned using flex properties.

1
2
3

.container {
  display: flex; /* enables Flexbox layout */
}

Example: The boxes above align horizontally by default since Flexbox uses flex-direction: row; as default.

2.2 flex-direction

The flex-direction property defines the main axis — determining whether items are arranged horizontally (row) or vertically (column), and in which order.

  • row – Default; left → right
  • row-reverse – right → left
  • column – top → bottom
  • column-reverse – bottom → top

2.3 flex-wrap

Controls whether flex items are forced into a single line or can wrap onto multiple lines.

  • nowrap – All items stay in one line (default).
  • wrap – Items wrap onto multiple lines (top → bottom).
  • wrap-reverse – Items wrap from bottom → top.

3. CSS Grid Layout

The CSS Grid Layout is a two-dimensional layout system that allows web designers to arrange elements into rows and columns easily. Unlike Flexbox, which works in a single direction (row or column), Grid gives full control over both rows and columns simultaneously — perfect for page or section layouts.

3.1 display: grid

To create a grid container, use display: grid; on a parent element. All direct children automatically become grid items.

1
2
3
4

.container {
  display: grid; /* enables grid layout */
}

Example: The boxes above are now part of a grid container.

3.2 grid-template-columns

Defines how many columns the grid should have and their widths.

  • px / % / fr – Fixed, percentage, or flexible fractions.
  • repeat() – Shorthand for repeating patterns.
  • auto-fit / auto-fill – Automatically fits items based on space.
