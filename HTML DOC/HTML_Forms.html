HTML Form Structure

Forms act a container for various form elements that allow users to input different types of data

Form submission: Form need to know where to and how to send data

Form Tag (<form>)

<form action="/submit" method="post"> </form>

Explanation:
The <form> tag is used to collect user input and send it to a server. It can contain fields like text boxes, radio buttons, checkboxes, dropdowns, and buttons.

  • action → URL where form data will be sent
  • method → HTTP method (GET or POST)
  • input → Used for various types of user input
  • label → Describes the input field
  • button → Used to submit/reset the form

Form Inputs

Form Input Types

1. Text Input

Used to enter normal single-line text such as name, city, or username.

<input type="text" placeholder="Enter your name">

2. Password Input

Masks the entered characters (●●●●) to keep the password private.

<input type="password" placeholder="Enter password">

3. Email Input

Accepts only valid email formats and shows email-specific keyboard on mobile.

<input type="email" placeholder="Enter your email">

4. Date Input

Allows users to pick a date using a calendar UI.

<input type="date">

5. Telephone Input

Used to enter phone numbers; shows numeric keypad on mobile.

<input type="tel" placeholder="Enter phone number">

6. Number Input

Accepts only numeric values; shows up/down arrows to change value.

<input type="number" min="1" max="100">

7. Checkbox

Used to select multiple options from a list.

<input type="checkbox" name="hobby" value="cricket"> Cricket

8. Radio Button

Used to select only one option from a group.

<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female

Important Input Attributes (Short Notes)

1. placeholder

Shows a light grey hint inside the input box until the user types.

<input type="text" placeholder="Enter your name">

2. required

Makes the field mandatory. Form cannot be submitted until filled.

<input type="email" required>

3. disabled

Makes the input inactive. User cannot click or edit it. Not submitted to server.

<input type="text" disabled>

4. readonly

Input is visible but cannot be edited. Value is still submitted.

<input type="text" value="India" readonly>

5. value

Sets a default or prefilled value inside the field.

<input type="text" value="Lakshman">

6. name

Key used to send data to the server. Required for form submission.

<input type="email" name="userEmail">

7. id

Gives unique identity to connect with label or JavaScript.

<input type="password" id="pwd">

8. min / max

Sets minimum and maximum limits for numbers or dates.

<input type="number" min="1" max="10">

9. maxlength

Limits how many characters the user can type.

<input type="text" maxlength="10">

10. pattern

Validates input using Regex rules.

<input type="tel" pattern="[0-9]{10}">

11. step

Defines step size for numbers or range sliders.

<input type="number" step="5">

12. autocomplete

Enables or disables browser suggestions for saved data.

<input type="text" autocomplete="off">

13. autofocus

Automatically focuses the input when the page loads.

<input type="text" autofocus>

Textarea Element

What is <textarea>?

The textarea element is used to enter multi-line text such as comments, messages, feedback, and descriptions.

Basic Example

<textarea></textarea>

Common Attributes

1. placeholder

Shows a hint inside the textarea before the user types.

<textarea placeholder="Enter your message"></textarea>

2. rows

Controls the height of the textarea (number of visible text lines).

<textarea rows="5"></textarea>

3. cols

Controls the width of the textarea (number of visible characters per line).

<textarea cols="40"></textarea>

4. maxlength

Sets the maximum number of characters allowed.

<textarea maxlength="200"></textarea>

5. readonly

Displays content but does not allow editing. Value is still submitted.

<textarea readonly>This can't be edited.</textarea>

6. disabled

Disables the textarea completely. Not submitted to the server.

<textarea disabled>Disabled textarea</textarea>

7. required

Makes the textarea mandatory before submitting a form.

<textarea required></textarea>

8. name

Used to identify textarea data during form submission.

<textarea name="userMessage"></textarea>

9. autofocus

Automatically focuses the textarea on page load.

<textarea autofocus></textarea>

Dropdown / Select Input

What is a Dropdown?

A dropdown (using the <select> tag) allows the user to choose ONE option from a list of predefined choices.

Example: Pick a Language

This dropdown lets the user select their preferred programming language.

<label for="lang">Choose a Language:</label>
<select id="lang" name="language">
  <option value="">-- Select Language --</option>
  <option value="html">HTML</option>
  <option value="css">CSS</option>
  <option value="js">JavaScript</option>
  <option value="python">Python</option>
  <option value="java">Java</option>
</select>

Dropdown Notes

  • <select> → Creates the dropdown
  • <option> → Each choice inside the dropdown
  • value="" → Data sent to server when the option is selected
  • First option can be used as a placeholder
