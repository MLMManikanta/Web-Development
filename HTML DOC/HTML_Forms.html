HTML Form Structure

Forms act as containers for form elements that allow users to input various types of data.

Form Submission: Forms need to know where and how to send the data.

Form Tag (<form>)

<form action="/submit" method="post"></form>

Explanation:
The <form> tag is used to collect user input and send it to a server. It can contain text fields, radio buttons, checkboxes, dropdowns, and buttons.

  • action → URL where form data will be sent
  • method → HTTP method (GET or POST)
  • input → Used for user input fields
  • label → Describes the input field
  • button → Used to submit or reset forms

Form Inputs

Form Input Types

1. Text Input

Used for entering normal single-line text (e.g., name, city, username).

<input type="text" placeholder="Enter your name">

2. Password Input

Masks entered characters (●●●●) to keep them private.

<input type="password" placeholder="Enter password">

3. Email Input

Accepts only valid email formats.

<input type="email" placeholder="Enter your email">

4. Date Input

Lets users pick a date using a calendar.

<input type="date">

5. Telephone Input

Used to enter phone numbers.

<input type="tel" placeholder="Enter phone number">

6. Number Input

Accepts only numeric values.

<input type="number" min="1" max="100">

7. Checkbox

Select multiple options from a list.

<input type="checkbox" name="hobby" value="cricket"> Cricket

8. Radio Button

Select only one option from a group.

<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female

Important Input Attributes (Short Notes)

1. placeholder

Displays a hint inside the input field until the user types.

<input type="text" placeholder="Enter your name">

2. required

Makes the field mandatory.

<input type="email" required>

3. disabled

Makes the input inactive and unclickable.

<input type="text" disabled>

4. readonly

Field is visible but cannot be edited (still submitted).

<input type="text" value="India" readonly>

5. value

Sets a default or pre-filled value.

<input type="text" value="Lakshman">

6. name

Specifies the key name for form submission.

<input type="email" name="userEmail">

7. id

Gives a unique identity (for label or JavaScript).

<input type="password" id="pwd">

8. min / max

Defines limits for number or date inputs.

<input type="number" min="1" max="10">

9. maxlength

Limits the number of characters allowed.

<input type="text" maxlength="10">

10. pattern

Validates input using regex patterns.

<input type="tel" pattern="[0-9]{10}">

11. step

Sets the increment value for numeric inputs.

<input type="number" step="5">

12. autocomplete

Enables or disables browser-saved suggestions.

<input type="text" autocomplete="off">

13. autofocus

Automatically focuses the input on page load.

<input type="text" autofocus>

Textarea Element

The <textarea> tag is used for multi-line text input (e.g., comments, messages, feedback).

Example:

<textarea placeholder="Enter your message"></textarea>

Dropdown / Select Input

The <select> tag creates a dropdown for selecting one option.

Example:

<select name="language">
  <option value="">-- Select Language --</option>
  <option value="html">HTML</option>
  <option value="css">CSS</option>
  <option value="js">JavaScript</option>
  <option value="python">Python</option>
</select>

About the <button> Tag

The <button> element creates clickable buttons.

  • button → Normal button (no form submission)
  • submit → Submits the form
  • reset → Clears all form fields
