Most Used Attributes

  • id attribute – used to uniquely identify an element.
    Example: <h2 id="heading1">Welcome</h2>
  • class attribute – used to group elements for styling or scripting.
    Example: <p class="info-text">This is a paragraph.</p>
  • style attribute – used to apply inline CSS to an element.
    Example: <h3 style="color: blue;">Styled Heading</h3>
  • title attribute – used to show extra information when hovered.
    Example: <span title="Hello!">Hover over me</span>
