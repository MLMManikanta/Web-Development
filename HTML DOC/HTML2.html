Most Used Attributes

  • id attribute – used to uniquely identify an element.
    Example: <h2 id="heading1">Welcome</h2>
  • class attribute – used to group elements for styling or scripting.
    Example: <p class="info-text">This is a paragraph.</p>
  • style attribute – used to apply inline CSS to an element.
    Example: <h3 style="color: blue;">Styled Heading</h3>
  • title attribute – used to show extra information when hovered.
    Example: <span title="Hello!">Hover over me</span>

Scientific & Formatting HTML Tags

1. Superscript (<sup>)

Definition: Superscript is used to display small text above the normal line.
Example: X2 + Y3 = 25

2. Subscript (<sub>)

Definition: Subscript is used for text written below the normal line (common in chemistry).
Example: H2O + CO2 = Carbonated Water

3. Figure & Figcaption (<figure> <figcaption>)

Definition: The <figure> tag groups visual content like images, diagrams with a caption using <figcaption>.

More Text Formatting Tags in HTML

4. Bold Text

Definition: Makes text bold.
Example: This is bold text

5. Italic Text

Definition: Makes text italic.
Example: This is italic text

6. Underline

Definition: Underlines the text.
Example: This is underlined text

7. Mark (Highlight)

Definition: Highlights text with yellow background.
Example: This is highlighted

8. Small Text

Definition: Displays small-sized text.
Example: This is small text

9. Deleted Text (<del>)

Definition: Shows text that is deleted (strikethrough).
Example: Old Price ₹150

10. Inserted Text (<ins>)

Definition: Shows newly added text (underlined).
Example: New Price ₹120

11. Code Tag

Definition: Used to display code snippets.
Example: console.log("Hello World");

12. Preformatted Text (<pre>)

Definition: Displays text exactly as typed (keeps spaces & new lines).
Example: 

Line 1
    Line 2 (indented)
        Line 3 (more indented)
