Most Used Attributes

  • id attribute – uniquely identifies an element.
    Example: <h2 id="heading1">Welcome</h2>
  • class attribute – groups elements for styling or scripting.
    Example: <p class="info-text">This is a paragraph.</p>
  • style attribute – applies inline CSS to an element.
    Example: <h3 style="color: blue;">Styled Heading</h3>
  • title attribute – shows extra info when hovered.
    Example: <span title="Hello!">Hover over me</span>

<details> and <summary> Tags

Explanation:

  • <details> creates a collapsible (expand/collapse) section.
  • <summary> defines the heading that users click to toggle the section.
  • Commonly used for FAQs, documentation, or hidden explanations.
  • When opened, the browser automatically adds an open attribute.

Example Code:

<details>
  <summary>Click to view more information</summary>
  <p>This section contains additional details that can be shown or hidden.</p>
</details>

Rendered Output:

Click to view more information

This section contains additional details that can be shown or hidden.

<dialog> Tag

Explanation:

  • <dialog> defines a popup window or dialog box.
  • It can contain text, images, or forms inside it.
  • Adding the open attribute makes it visible by default.
  • It can also be controlled dynamically using JavaScript (show() / close()).

Example Code:

<dialog open>
  <p>This is an open dialog window centered on the page.</p>
</dialog>

Rendered Output:

This is an open dialog window centered on the page.
