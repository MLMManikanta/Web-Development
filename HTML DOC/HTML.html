HTML Basic Concepts

What is HTML?

  • HTML – Hyper Text Markup Language.
    Example: <html>...</html>
  • Hyper Text – Linking between web pages.
    Example: <a href="page.html">Go</a>
  • Markup Language – Uses special syntax to structure content like headings, images, paragraphs.
    Example: <h1>Heading</h1>

HTML Boilerplate

  • Shortcut: Type ! and press Enter to generate boilerplate.
  • Nesting: Elements placed inside each other (HTML → Head & Body).
    Example: <body> <h1>Hello</h1> </body>
  • Indenting: Organized spacing to show hierarchy clearly.

Boilerplate Code

  • <!DOCTYPE html> – Declares HTML5 document.
    Example: <!DOCTYPE html>
  • <html lang="en"> – Root element with language set to English.
    Example: <html lang="en">
  • <head> – Holds metadata like title, charset, styles.
    Example: <head>...</head>
  • <meta charset="UTF-8"> – Supports all characters.
    Example: <meta charset="UTF-8">
  • <meta name="viewport"> – Makes page responsive.
    Example: <meta name="viewport" content="width=device-width, initial-scale=1.0">
  • <title> – Shows title on browser tab.
    Example: <title>Home Page</title>
  • <link rel="icon"> – Adds favicon icon.
    Example: <link rel="icon" href="logo.png">

Heading Tags

  • <h1> – Main largest heading.
    Example: <h1>Main Title</h1>
  • <h2> – Sub-heading.
    Example: <h2>Sub Heading</h2>
  • <h3> – Section heading.
    Example: <h3>Section</h3>
  • <h4>, <h5>, <h6> – Smaller headings.
    Example: <h6>Smallest Heading</h6>

Paragraph & Line Break

  • <p> – Used for paragraph text.
    Example: <p>This is a paragraph.</p>
  • <br> – Inserts a line break.
    Example: Hello<br>World

Types of Lists in HTML

1. Ordered List (<ol>)

Explanation: An Ordered List displays items in a numbered order such as 1, 2, 3. Each item is written inside an <li> tag.

Example Code:

<ol>
    <li>HTML Basics</li>
    <li>CSS Styling</li>
    <li>JavaScript Concepts
        <ol>
            <li>Variables</li>
            <li>Functions</li>
            <li>Loops</li>
        </ol>
    </li>
    <li>React Framework</li>
</ol>
  1. HTML Basics
  2. CSS Styling
  3. JavaScript Concepts
    1. Variables
    2. Functions
    3. Loops
  4. React Framework

2. Unordered List (<ul>)

Explanation: An Unordered List displays items with bullet points. Useful for categories or unordered items.

Example Code:

<ul>
    <li>Frontend Technologies</li>
    <li>Backend Technologies</li>
    <li>Databases
        <ul>
            <li>MySQL</li>
            <li>MongoDB</li>
            <li>PostgreSQL</li>
        </ul>
    </li>
    <li>DevOps Tools</li>
</ul>
  • Frontend Technologies
  • Backend Technologies
  • Databases
    • MySQL
    • MongoDB
    • PostgreSQL
  • DevOps Tools

3. Description List (<dl>, <dt>, <dd>)

Explanation: A Description List contains terms and descriptions. Ideal for definitions, FAQs, or glossary sections.

Example Code:

<dl>
    <dt>HTML</dt>
    <dd>A markup language used to structure webpages.</dd>

    <dt>CSS</dt>
    <dd>Used for styling webpages with layouts and colors.</dd>

    <dt>JavaScript</dt>
    <dd>A scripting language for interactivity.</dd>
</dl>
HTML
A markup language used to structure webpages.
CSS
Used for styling webpages with layouts and colors.
JavaScript
A scripting language for interactivity.

Comments

  • HTML Comments – Notes not shown on the webpage.
    Example: <!-- This is a comment -->
  • Shortcut (Windows): Shift + Alt + A

File Paths

  • Relative Path – Starts from current file.
    Example: ./images/pic.png
  • Absolute Path – Full location or full URL.
    Example: https://example.com/image.jpg

Hyperlinks

  • <a> – Creates clickable links.
    Example: <a href="page.html">Go to Page</a>
  • Internal Link – Uses ID to jump within page.
    Example: <a href="#section">Go Down</a>

Image Tag (<img>)

Explanation:
The <img> tag is used to display images in a webpage. It is a self-closing tag and requires key attributes like:

  • src → Source of the image (file path or URL)
  • alt → Text shown when image fails to load (important for accessibility)
  • height → Sets the image height
  • width → Sets the image width

Additional Info:

  • Relative Path: Loads image from the current folder (e.g., ./images/logo.png)
  • Absolute Path: Full URL to an image on the internet
  • alt text improves accessibility and SEO
  • If only height is given, width adjusts automatically (and vice-versa)

Example Code:

<img src="./html logo.png" alt="HTML Logo" height="100px" width="100px">

SVG Image (<svg> / .svg file)

Explanation:
SVG (Scalable Vector Graphics) is a vector-based image format. Unlike PNG or JPG, SVG images never lose quality when zoomed in. They are ideal for logos, icons, charts, and shapes.

  • Scalable: Does not pixelate when enlarged
  • Editable: You can style SVG with CSS
  • Lightweight: File size is very small
  • Interactive: Can animate and use JavaScript inside

Example Code:

<svg width="120" height="120">
    <circle cx="60" cy="60" r="50" fill="orange" stroke="black" stroke-width="3"></circle>
</svg>

Bitmap Image (BMP Format)

Explanation:
BMP (Bitmap) is an uncompressed image format. It stores every pixel exactly, giving very high quality — but the file size becomes extremely large. Because of this, BMP is not recommended for websites.

  • High Quality: No compression applied
  • Very Large Size: Slows down webpages
  • Not Web Friendly: Browsers support it, but it's rarely used
  • Better Alternatives: JPG, PNG, WebP

Example Code:

<img 
    src="example.bmp" 
    alt="Bitmap Image Example" 
    width="150px">
