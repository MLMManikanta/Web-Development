HTML Basic Concepts

Understanding the foundation of web structure and content organization

What is HTML?

  • HTML – Hyper Text Markup Language.
    Example: <html>...</html>
  • Hyper Text – Linking between web pages.
    Example: <a href="page.html">Go</a>
  • Markup Language – Uses special syntax to structure content like headings, images, paragraphs.
    Example: <h1>Heading</h1>

HTML Boilerplate

  • Shortcut: Type ! and press Enter to generate boilerplate.
  • Nesting: Elements placed inside each other (HTML → Head & Body).
  • Indenting: Organized spacing to show hierarchy clearly.

Boilerplate Code

  • <!DOCTYPE html> – Declares HTML5 document.
  • <html lang="en"> – Root element with language set to English.
  • <head> – Holds metadata like title, charset, styles.
  • <meta charset="UTF-8"> – Supports all characters.
  • <meta name="viewport"> – Makes page responsive.
  • <title> – Shows title on browser tab.
  • <link rel="icon"> – Adds favicon icon.

Types of Lists in HTML

1. Ordered List (<ol>)

Explanation: Ordered lists display items in numbered order.

<ol>
    <li>HTML Basics</li>
    <li>CSS Styling</li>
</ol>
  1. HTML Basics
  2. CSS Styling
  3. JavaScript Concepts
    1. Variables
    2. Functions
    3. Loops
  4. React Framework

2. Unordered List (<ul>)

Explanation: Unordered lists use bullet points instead of numbers.

  • Frontend Technologies
  • Backend Technologies
  • Databases
    • MySQL
    • MongoDB
    • PostgreSQL
  • DevOps Tools

3. Description List (<dl>, <dt>, <dd>)

Explanation: Used for definitions or glossary items.

HTML
A markup language used to structure webpages.
CSS
Used for styling webpages with layouts and colors.
JavaScript
A scripting language for interactivity.

Image Tag (<img>)

Explanation: The <img> tag displays images in webpages.

  • src → Source of the image
  • alt → Alternate text (for accessibility)
  • height & width → Image size

HTML Landmark Tags

Landmark tags define page structure and improve accessibility.

  • <header> – Top section (logo/navigation)
  • <nav> – Contains site links
  • <main> – Main content area
  • <section> – Thematic grouping of content
  • <article> – Independent piece (blog, news)
  • <aside> – Related side info
  • <footer> – Bottom section (copyright)
