What is a Table?

An HTML table is used to display data in rows and columns. The table structure is defined using <table>, which contains table rows (<tr>) and table cells (<td> or <th>).

Table Elements

  • <table> → Defines the table container
  • <thead> → Header section
  • <tbody> → Body/content section
  • <tfoot> → Footer section
  • <tr> → Table row
  • <th> → Table header cell (bold & centered by default)
  • <td> → Table data cell

Evolution of Layouts in Web Design

  1. Table-based Layouts (1990s) – Used tables to design whole webpages. Heavy and not responsive.
  2. Float-based Layouts (2000s) – Used CSS float property; better but still limited.
  3. Flexbox Layout (2015+) – One-dimensional layout; easy alignment & spacing.
  4. CSS Grid Layout (2017+) – Two-dimensional layout; best for complex designs.
  5. Modern Responsive Layouts – Flexbox + Grid + Media Queries.
