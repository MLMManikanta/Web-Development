Inline JavaScript

Inline JavaScript is written directly inside an HTML attribute such as onclick.

Example

<button onclick="alert('Hello!')">Click Me</button>

Internal JavaScript

Internal JavaScript is written within a <script> tag inside the same HTML file.

Example

This text will be changed using Internal JS.

document.getElementById("internalText").style.color = "blue";

External JavaScript

External JavaScript is stored in a separate file and linked using the src attribute.

Example

This text will be updated by the external JS file.

<script src="external.JS" defer></script>

JavaScript in <head> (defer & async)

Using defer

Runs after the HTML document is fully parsed.

<script src="file.js" defer></script>

Using async

Loads independently and may execute before HTML is finished parsing.

<script src="file.js" async></script>

JavaScript Basics

Introduction

JavaScript Basics include variables, data types, operators, and expressions.

Variables

Variables store values. Use var, let, and const.

let name = "John";
const age = 25;
var city = "NYC";

Data Types

  • String
  • Number
  • Boolean
  • Array
  • Object
  • Undefined
  • Null

Operators

let x = 10;
let y = 5;
let sum = x + y;
