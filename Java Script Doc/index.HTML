JavaScript Tutorial

1. Inline JavaScript

Inline JavaScript is written directly inside an HTML tag using event attributes.


<button onclick="alert('Hello!')">
    Click Me
</button>

2. Internal JavaScript

Internal JavaScript is written inside a <script> tag within the same HTML file.

This text will be changed by Internal JavaScript.


<script>
    document.getElementById("internalText").style.color = "blue";
</script>

3. External JavaScript

External JavaScript is written in a separate file like external.JS.

This text will be changed by External JavaScript.


<script src="external.JS" defer></script>

--- external.JS file ---
document.querySelector(".externalJS").style.color = "green";

4. JavaScript in <head> using defer & async

defer: Runs after the HTML is fully loaded.

async: Loads independently, useful for analytics.


<script src="file.js" defer></script>
<script src="file.js" async></script>
